# Session – 2026-05-19

## Topics covered
- KPI dashboards: writing a "KPI contract" before any query (business question, definition, edge cases, unit, misleading factors)
- Recognising and fixing bad KPIs: meaningless averages, type errors, status-blind counts
- Conditional aggregation: `COUNT(CASE WHEN ... THEN 1 END)` and `SUM(CASE WHEN ...)` to pivot status groups into columns
- Safe division with `NULLIF` to avoid divide-by-zero
- `PERCENTILE_CONT(0.5) WITHIN GROUP (ORDER BY ...)` for median resolution time
- `ROLLUP` to add a totals row to a summary query
- Window function `AVG(...) OVER ()` to flag rows below the overall average in the same SELECT
- Urgency scoring: translating VARCHAR priority into numeric weights with `CASE` for arithmetic expressions
- SLA target validation inline with a nested `CASE` inside `WHEN`
- LEFT JOIN chain (teams → users → tasks) to keep zero-task teams in the result
